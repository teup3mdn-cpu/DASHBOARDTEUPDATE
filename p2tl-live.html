Konfigurasi Koneksi Google Sheets

Pilih metode integrasi data yang Anda inginkan

KELEBIHAN METODE PRIVAT (REKOMENDASI):

  • Spreadsheet Anda **TETAP PRIVAT (KUNCI)** - tidak perlu dipublikasikan ke web umum.
  • Hanya membutuhkan **1 Link Web App URL saja** (Sistem akan menarik data Target & Realisasi secara otomatis).
  • Keamanan terlindungi penuh dengan kode **Token Keamanan Rahasia**.

Dapatkan URL ini setelah menerapkan (deploy) kode script pada Google Sheets Anda.

Panduan Penting Koneksi CSV Publik:

Metode ini mewajibkan Anda untuk membagikan file ke publik dengan format ekspor CSV pada masing-masing tab:

  1. Pilih tab pencapaian kwh p2tl untuk Target, dan RUMUS REALISASI untuk Realisasi.
  2. Wajib mengubah pilihan dari "Halaman Web" menjadi Nilai Terpisahkan Koma (.csv).

CARA MENJALANKAN LOCAL HOST (PILIH SALAH SATU):

Metode 1: Menggunakan Terminal CMD (Python)

Buka CMD pada folder folder file HTML ini disimpan, lalu jalankan perintah:

python -m http.server 8000
Metode 2: Menggunakan Live Server di VS Code

Buka file di VS Code, instal ekstensinya, lalu klik "Go Live".
